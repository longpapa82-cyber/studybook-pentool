## 🎯 테스트 방법

### 키보드 단축키 테스트
1. PDF 업로드
2. 키보드로 도구 전환:
   - `p` → 펜 도구
   - `h` → 형광펜
   - `e` → 지우개
   - `v` → 선택 모드
3. 줌 단축키 테스트:
   - `Ctrl++` → 확대
   - `Ctrl+-` → 축소
   - `Ctrl+0` → 리셋

### 포인트 단순화 테스트
1. PDF에서 긴 선 그리기
2. 개발자 도구 콘솔에서 확인:
